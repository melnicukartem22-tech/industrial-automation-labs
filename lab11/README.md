# Лабораторна робота №11

## Тема

Основи роботи з базами даних та SQL. Робота з SQL в Node-RED.

## Мета роботи

Ознайомитися з основами роботи СКБД MariaDB, навчитися створювати та змінювати структуру баз даних і таблиць за допомогою SQL-запитів, отримати практичні навички використання баз даних у Node-RED для збереження та обробки інформації.

## Висновок

У ході виконання лабораторної роботи було отримано практичні навички роботи з СКБД MariaDB та клієнтом HeidiSQL. Було створено бази даних, таблиці, виконано операції додавання, редагування, вибірки та видалення даних за допомогою SQL-запитів. Також було освоєно використання Node-RED для роботи з базами даних MariaDB, створення автоматизованих потоків збору інформації про ресурси комп’ютера та збереження цих даних у базу даних. Результатом роботи стала система моніторингу ресурсів ПК із відображенням інформації на Dashboard та збереженням історичних даних у MariaDB.
