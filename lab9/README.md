# 2. Створення клієнта IPAPI у Node-RED

Для отримання інформації через HTTP API було створено потік у Node-RED.

Структура потоку:

```text
Inject → HTTP Request → Debug
```

У вузлі HTTP Request використовувався URL:

```http
http://ip-api.com/json/
```

Після натискання кнопки Inject було отримано відповідь від сервера та виведено її у вікно Debug.

Скріншот роботи потоку наведено на рисунку 3.

![Рисунок 3](media/1(3).png)

Отримані дані містять інформацію про поточне розташування користувача та його інтернет-провайдера.

# 3. Отримання погодних даних за допомогою Open-Meteo

Для отримання прогнозу погоди використано сервіс Open-Meteo.

У вузлі HTTP Request був сформований запит:

```http
https://api.open-meteo.com/v1/forecast?latitude=50.45&longitude=30.52&hourly=temperature_2m
```

де:

* latitude = 50.45;
* longitude = 30.52;
* hourly = temperature_2m.

Для візуалізації даних було створено потік:

```text
Inject → HTTP Request → Function → Chart
```

Скріншот потоку наведений на рисунку 4.

![Рисунок 4](media/1(4).png)

# 4. Обробка даних у вузлі Function

Для перетворення JSON-даних у формат, який підтримує Chart, використано наступний код:

```javascript
let data = [];

for (let i = 0; i < msg.payload.hourly.time.length; i++) {
    data[i] = {
        x: new Date(msg.payload.hourly.time[i]),
        y: msg.payload.hourly.temperature_2m[i]
    };
}

msg.payload = data;
return msg;
```
