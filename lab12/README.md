
# Лабораторна робота №12

**Виконав:** студент групи КС 1-2

**ПІБ:** Мельничук Артем Євгенійович

**Спеціальність:** Автоматизація, комп'ютерні технології та робототехніка  

## Тема

Інтегрування з хмарними застосунками та сервісами

## Мета роботи

Набуття практичних навичок інтеграції локальних систем автоматизації та середовища Node-RED із глобальними хмарними застосунками та сервісами. Освоєння методів взаємодії з API Google Sheets для реєстрації технологічних даних та розробка людино-машинного інтерфейсу оператора у вигляді Telegram-бота.

## Лабораторна установка

- Персональний комп’ютер або віртуальна машина.
- Docker.
- Node-RED.
- Google Sheets API.
- Telegram Bot API.
- Git та GitHub.
